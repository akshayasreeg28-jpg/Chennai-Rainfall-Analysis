# Chennai-Rainfall-Analysis
Exploratory analysis of rainfall patterns in Chennai (2000–2025)

Objective:

To analyze long-term rainfall patterns in Chennai and identify seasonal trends and extreme rainfall events.

Tools Used:

Python,
Pandas,
Matplotlib,
Seaborn.

Key Analysis Performed:

Data cleaning & preprocessing,
Feature engineering (Year & Month extraction),
Year-wise rainfall trend,
Monthly rainfall distribution,
Seasonal pattern analysis,
Extreme rainfall detection.

Key Insights

Rainfall is highly seasonal, concentrated during Northeast Monsoon (Oct–Nov).
November records the highest average rainfall.
Significant variability exists across years.
Extreme rainfall events contribute heavily to annual totals.
